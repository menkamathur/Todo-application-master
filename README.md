# ✅ ToDo Application

This is a simple ToDo web app made using HTML, CSS, JavaScript, and JSON Server.

---

## 🚀 Features

- Add tasks (title + description)
- Delete tasks
- Mark tasks as done
- Responsive UI
- Backend with JSON Server

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- jQuery  
- JSON Server
